Terminalo

Terminalo is a JS library designed to provide useful terminal lines to your site
with copy function with little effort. You can use Terminalo two ways:

Download

It is just one file! Get it here:

Usage

1. Block

Use it when you have multiple commands. Each line takes full width of wrapper.
Only basic styles are provided, further customization is up to you.

Feel free to click buttons below. These commands will download the
repository to your computer.

git clone https://github.com/bouncepaw/terminalo.git
cd terminalo
<!-- What to write: -->
<div class="terminalo">
  <div class="terminalo__line">
    git clone https://github.com/bouncepaw/terminalo.git
  </div>
  <div class="terminalo__line">cd terminalo</div>
</div>

<!-- It transformes into: -->
<div class="terminalo">
  <div class="terminalo__line">
    <button class="terminalo__button" id="terminalo__button_0">^C</button>
    <span class="terminalo__cmd" id="terminalo__cmd_0">
      git clone https://github.com/bouncepaw/terminalo.git
    </span>
  </div>
  <div class="terminalo__line">
    <button class="terminalo__button" id="terminalo__button_1">^C</button>
    <span class="terminalo__cmd" id="terminalo__cmd_1">
      cd terminalo
    </span>
  </div>
</div>

2. Inline

Sometimes you need one little inline command. In this case use
Terminalo without wrapper: sudo rm -rf /*

<!-- What to write: -->
<span class="terminalo__line">sudo rm -rf /*</span>

<!-- It transformes into: -->
<span class="terminalo__line">
  <button class="terminalo__button" id="terminalo__button_2">^C</button>
  <span class="terminalo__cmd" id="terminalo__cmd_2">sudo rm -rf /*</span>
</span>
<!-- Note that each Terminalo button and command on page has unique class.-->
